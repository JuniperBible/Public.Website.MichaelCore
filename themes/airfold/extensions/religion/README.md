# Religion Extension

Hugo extension for displaying Bible texts, commentaries, and dictionaries from SWORD/e-Sword modules.

## Features

- **Bible Display** - Verse-by-verse reading with Strong's numbers
- **Chapter Navigation** - Previous/next chapter links
- **Book Overview** - Chapter listing for each book
- **Translation List** - Browse available Bible translations

## Directory Structure

```
extensions/religion/
├── content/
│   └── bibles/
│       └── _content.gotmpl    # Generates pages from JSON
├── layouts/
│   └── bibles/
│       ├── list.html          # Bible list page
│       └── single.html        # Bible/book/chapter pages
└── README.md
```

## Data Files

This extension requires JSON data files generated by the juniper tool:

| File | Description |
|------|-------------|
| `data/bibles.json` | Bible metadata (id, title, language, features) |
| `data/bibles_auxiliary.json` | Full content (books, chapters, verses) |

### Generate Data

```bash
cd tools/juniper
go build -o juniper ./cmd/juniper
./juniper migrate --source ~/.sword
./juniper convert
```

## Hugo Configuration

Add to `hugo.toml`:

```toml
# Module mounts for religion content
[[module.mounts]]
  source = "themes/airfold/extensions/religion/content/bibles"
  target = "content/religion/bibles"

[[module.mounts]]
  source = "themes/airfold/extensions/religion/layouts/bibles"
  target = "layouts/religion/bibles"

# UI strings (optional)
[params.ui]
  bibles = "Bibles"
  biblesDescription = "Browse available Bible translations"
  backToReligion = "Back to Religion"
  backToBible = "Back to Bible"
  backToBook = "Back to Book"
  allBibles = "All Bibles"
  allChapters = "All Chapters"
```

## URL Structure

Pages are generated at:

- `/religion/bibles/` - List of all Bibles
- `/religion/bibles/{bible}/` - Bible overview with book list
- `/religion/bibles/{bible}/{book}/` - Book with chapter list
- `/religion/bibles/{bible}/{book}/{chapter}/` - Chapter with verses

## Styling

The extension uses AirFold theme classes:

- `paper-card` - Card styling for Bible list
- `btn-paper`, `btn-paper-dark` - Navigation buttons
- `prose-paper` - Content typography
- `bible-text` - Verse text styling

Add custom CSS for verse numbers and Strong's numbers:

```css
.bible-text strong {
  color: var(--color-accent);
  font-weight: normal;
  font-size: 0.75em;
  vertical-align: super;
}

.bible-text sup {
  color: var(--color-paper-gray);
  font-size: 0.6em;
}
```
