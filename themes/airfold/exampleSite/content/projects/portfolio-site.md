## Overview

A personal portfolio website showcasing my work, blog posts, and professional credentials.

## Goals

- Establish an online presence
- Share knowledge through blog posts
- Display certifications and skills
- Provide a contact mechanism

## Implementation

Built using the AirFold theme with all extensions enabled:
- Blog extension for articles
- Portfolio extension for projects
- Certifications extension for credentials
- Resume extension for CV

## Results

A clean, fast, and accessible personal website that works great on all devices.
