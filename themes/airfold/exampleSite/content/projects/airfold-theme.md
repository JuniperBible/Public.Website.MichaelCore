## Overview

AirFold is a Hugo theme that brings a handwritten, paper notebook aesthetic to your website. Built with Tailwind CSS v4.

## Features

- Paper-style cards with wavy borders
- Full dark mode support
- Responsive design
- SEO optimized
- Modular extensions

## Tech Stack

- Hugo static site generator
- Tailwind CSS v4
- Vanilla JavaScript
- OpenPGP.js for contact encryption
